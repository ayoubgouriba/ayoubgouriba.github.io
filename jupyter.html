Downtown Ottawa Walking Street Network Analysis

Link to the full report (html) available here.

In my GISY6060 Assignment 5 project, I conducted an in-depth analysis of the walking street network in downtown Ottawa, specifically focusing on the Somerset neighborhood. Utilizing Python libraries such as Pandas, Geopandas, OSMnx, NetworkX, and Shapely, I manipulated and visualized street network data to reveal insights into the area. The project encompassed visualizing layers of data, including buildings, points of interest, and parks, generating isochrones, and computing key street network indicators like basic measures, density-based statistics, and betweenness centrality.
