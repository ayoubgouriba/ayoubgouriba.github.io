Walking itineraries web map application

Ottawalk is a web map application that generates personalized walking itineraries in Ottawa based on user-selected profiles such as nature, culture, or market. Built using open-source data and tools: OSM, PostgreSQL, GeoServer, Leaflet.

Estimating Solar Energy Potential in Ottawa's Sandy Hill Neighborhood

Determining the solar power potential of the Sandy Hill neighborhood in Ottawa, Canada. Spatial analyst tools in ArcGIS Pro were used, specifically the Area Solar Radiation tool, to calculate the amount of solar radiation received by each suitable rooftop in the neighborhood.

Downtown Ottawa Walking Street Network Analysis

In this Jupyter Notebook, downtown Ottawa's walking street network is analyzed using libraries like Pandas, Geopandas, OSMnx, NetworkX, and Shapely. Data layers are visualized, and street network indicators, including density-based statistics and betweenness centrality, are calculated.

Forest Types & Characteristics in Lawrencetown, NS:
Hemlock, Black and White Spruce

This map has been produced using ArcGIS Pro as a part of the requirements for the Geographic Information Systems Certificate Program at the Centre of Geographic Sciences, NSCC, Lawrencetown, Nova Scotia.

Board Games Exploratory Data Analysis & Predictive Modeling

Exploratory data analysis of a webscraped database from the BoardGameGeek.com website. The focus of the analysis is to build a predictive model using machine learning algorithms to predict the difficulty of games.
