Board Games Exploratory Data Analysis & Predictive Modeling

Link to the full report (pdf) available here.

In this comprehensive study, I delve into the world of board games using a web-scraped dataset from BoardGameGeek.com. The primary objective of this project is to perform an exploratory data analysis to identify patterns and relationships within the dataset, and ultimately build a predictive model using machine learning algorithms to predict the complexity of board games. With a dataset containing 20,343 data points and 14 variables, including game attributes such as ratings, game length, and mechanics, I employ R to carry out the analysis and modeling process.

The project involves several key steps, such as data preparation, exploratory data analysis, and predictive modeling. I construct three machine learning models using 10-fold cross-validation on an 80/20 split, testing Decision Tree, Random Forest, and K-Nearest Neighbors (KNN) algorithms. By comparing their performance, I aim to determine the most accurate and effective model to predict a game's complexity based on its attributes.
